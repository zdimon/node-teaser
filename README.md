#Exercise Robot Teaser

## Installation

	git clone git@github.com:zdimon/node-teaser.git
	cd node-teaser
	npm install
	cd public
	ln -s ../node_modules node_modules


## Running in prod mode

	npm starrun prodserver

## Running in dev mode

	npm run devserver


## Checking

	http://localhost:3000
